HTML (HyperText Markup Language) is the core syntax for putting information on the worldwide web. If you want to create or modify web pages, it's best if you learn HTML.

The links on this page lead to a variety of HTML tutorials and HTML training materials. Whether you are just starting out, learning the basics of HTML, or are an old hand at web development wanting to learn HTML5 (the newest version of the standard), you can find helpful resources here for HTML best practices.
